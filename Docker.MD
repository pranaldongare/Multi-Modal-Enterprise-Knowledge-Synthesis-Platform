# Docker
## `make build`
This command will build the Docker image for the project and pull the MongoDB image from Docker Hub.
- run this if you ever change code

## `make ollama`
This command will start two instances of Ollama for parallel processing.

## `make run`
This command will start the FastAPI server and the frontend server and everything elese needed to run the project.

rename `.env.docker` to `.env` before running the docker
